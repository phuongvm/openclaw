# OpenClaw — Personal AI Assistant

Personal AI assistant running on your own devices. Multi-channel: WhatsApp, Telegram, Slack, Discord, Google Chat, Signal, iMessage, Teams, WebChat, BlueBubbles, Matrix, Zalo. Gateway = control plane; extensions in extensions/; UI in ui/; apps (Swift/Kotlin) in apps/. Node ≥22, npm/pnpm/bun. CLI: openclaw onboard, openclaw gateway, openclaw agent.
